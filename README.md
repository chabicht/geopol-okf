# Geopolitics Open Knowledge Bundle

A structured knowledge base covering global geopolitics — actors, conflicts,
regions, themes, and events — synthesized from Substack essays, news analysis,
and strategic assessments.

## Directories

* [Actors](actors/) — Countries, leaders, organizations, and analysts whose work shapes the discourse
* [Conflicts](conflicts/) — Wars, crises, and sustained standoffs
* [Regions](regions/) — Geographic areas with independent dynamics
* [Themes](themes/) — Cross-cutting analytical topics and frameworks
* [Events](events/) — Discrete occurrences
